# VibeBoyRunner — GitHub Pages

Published artifacts for the [VibeBoyRunner](https://github.com/vibeboyrunner/vibeboyrunner) project.

## Structure

### `setups/`

Rendered setup scripts and agent skills, published by CI on each version tag.

| Path | Description |
|------|-------------|
| `setups/<tag>/setup.sh` | Version-pinned setup script |
| `setups/<tag>/setup_skill.md` | Agent skill for guided setup |
| `setups/latest/` | Always matches the most recent release |
